# 🌐 Akhil Reddy — Personal Portfolio

> My personal portfolio website built with HTML, CSS, and JavaScript — showcasing my projects, skills, and how to get in touch.

🔗 **Live Site:** [akhil2699.github.io](https://akhil2699.github.io)

---

## ✨ Features

- 🌙 **Dark / Light Mode** toggle
- 🎬 **Smooth scroll animations** — sections fade in as you scroll
- ⌨️ **Typewriter effect** in the hero section
- 💎 **Glassmorphism cards** for projects and skills
- 📱 **Fully responsive** — works on mobile, tablet, and desktop
- ⚡ **No frameworks** — pure HTML, CSS & JavaScript

---

## 📂 Sections

| Section | Description |
|---|---|
| **Hero** | Introduction with animated typing effect and CTA buttons |
| **About** | A short bio and stats about me |
| **Projects** | Cards for each project with tech tags and live links |
| **Skills** | Frontend, Backend, Tools, and what I'm currently learning |
| **Contact** | Email, GitHub, and LinkedIn links |

## 📦 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/Akhil2699/akhil2699.github.io.git

# 2. Open the folder
cd akhil2699.github.io

# 3. Open in your browser
# Just double-click index.html  OR  use VS Code Live Server
```

No build step, no dependencies — it just works.

## 🔄 Deployment

This site is deployed automatically via **GitHub Pages**.

Every push to the `main` branch triggers a redeployment. The live site updates within ~1 minute.
